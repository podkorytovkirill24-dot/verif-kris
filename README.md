# Telegram бот для верификации номеров (Crypto Pay)

## Что добавлено
- Авто-проверка оплаты (без кнопки): бот сам подтверждает оплату и кидает заявку в очередь.
- Поиск пользователя по `@username`.
- История админ-действий (аудит: кто/что/когда).
- Бан/разбан по `@username`.
- Экспорт очереди в `.csv`.

## Админ-панель
- `Пользователи`
- `Поиск user`
- `Очередь`
- `Экспорт CSV`
- `История`
- `Статистика`
- `Выдать/убрать админку`
- `Выдать/убрать free`
- `Выдать/убрать приоритет`
- `Бан/Разбан`

## Установка
```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## Настройка
1. Скопируй `.env.example` в `.env`.
2. Заполни:
- `BOT_TOKEN`
- `SUPERADMINS` (username без `@`, через запятую)
- `OPERATOR_PRICES`
- `CRYPTO_PAY_API_TOKEN`
- `CRYPTO_PAY_TESTNET`
- `CRYPTO_PAY_ACCEPTED_ASSETS`
- `AUTO_PAY_CHECK_INTERVAL`

## Запуск
```bash
python bot.py
```

## Важно
- Все права и ограничения выдаются по `@username`.
- Если у пользователя нет username, он не сможет полноценно работать с системой.
- Админские уведомления приходят тем админам, кто уже нажал `/start` боту.
